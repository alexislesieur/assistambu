# AssistAmbu

Outil de gestion pour équipes ambulancières.

## Architecture

| Dossier | Rôle | Stack |
|---------|------|-------|
| `Api/`  | API REST | Laravel (PHP) |
| `App/`  | Interface principale | Next.js (TypeScript) |
| `Auth/` | Authentification | Next.js (TypeScript) |

## Lancement

```bash
./dev.sh
```

## Prérequis

- PHP 8.x + Composer
- Node.js 18+
- Docker (pour la base de données)
